Crystals Collector Game

Ready?

You'll be given a target number to match during each round. Click on the crystals below to add some points to your score.

Be careful!

Each crystal is assigned a random number, so you'll need to be vigilant in calculating how many points each crystal is adding to your total score. If you match the target number, you win the round! If you go over, you lose that round.

Things be a-changin'

The value of each crystal is reset each round, so be prepared to do some mental math!
